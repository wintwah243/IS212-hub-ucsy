# Other Types of Mining (slide15 - 8.6.2026)

While we have discussed general data mining, this slide focuses on specialized techniques used for specific types of information, particularly Text Mining, and lists other advanced functions.

---

## 1. Text Mining

Text mining is the application of data mining techniques specifically to textual documents (like emails, articles, or reports) rather than numbers in a table. It includes several key tasks:

    Text mining ကဇယားကွက်တစ်ခုအတွင်းရှိတဲ့ ကိန်းဂဏန်း (Numbers) များကို ဆန်းစစ်မဲ့အစား၊ စာသားများဖြင့် ဖွဲ့စည်းထားသော စာရွက်စာတမ်းများ (Textual documents) ကို data mining နည်းစနစ်များ အသုံးပြု၍ ရှာဖွေခြင်း ဖြစ်တယ်။ (ဉပမာ-email, article, report)

---

## 2. Sentiment Analysis:

**What it is:** Using computers to predict the "feeling" behind a piece of writing.

    ကွန်ပျူတာ/AI ကို အသုံးပြုပြီး လူတွေ ရေးသားထားတဲ့ စာသားတွေရဲ့ နောက်ကွယ်က "ခံစားချက်/စိတ်နေသဘောထား (Feeling or Sentiment)" ကို အလိုအလျောက် ခန့်မှန်းတွက်ချက်ခိုင်းတဲ့ နည်းပညာ

**Example:** Learning to automatically predict if a customer review is positive or negative about a product

---

## 3. Information Extraction:

What it is: Taking "unstructured" text (sentences and paragraphs) and turning it into "structured" data (like a neatly organized table).

    လူတွေ ရေးထားတဲ့ ဖတ်ရုံပဲရမယ့် "Unstructured" Text (ပုံသေဖွဲ့စည်းထားခြင်းမရှိတဲ့ စာကြောင်းတွေ၊ စာပိုဒ်တွေ) ထဲကနေ ကွန်ပျူတာက တွက်ချက်လို့ရမယ့် "Structured" Data (သေသပ်စွာ ဖွဲ့စည်းထားတဲ့ ဇယားကွက်ပုံစံ) အဖြစ်သို့ ပြောင်းလဲထုတ်ယူပေးတဲ့ နည်းပညာ။ ဥပမာအားဖြင့် အီးမေးလ်စာသား
    ဝင်လာမယ် -
    "Wint Wah က မနက်ဖြန် ညနေ ၃ နာရီမှာ ရန်ကုန်မြို့က ဆရာမနဲ့ တွေ့ဆုံဖို့ ချိန်းဆိုထားပါတယ်"
    Information Extraction စနစ်က ဒီစာကို ဖတ်ပြီး အလိုအလျောက် ဇယားကွက် ထုတ်ပေးလိုက်တာမျိုးပါ -

## 4. Entity Recognition and Disambiguation:

**What it is:** Identifying specific people, places, or things in text and making sure the computer knows exactly which one is being discussed.

    စာသားတွေထဲကနေ သီးသန့်ဖြစ်တဲ့ လူနာမည် (People)၊ နေရာဒေသ (Places)၊ သို့မဟုတ် ကုမ္ပဏီ/ပစ္စည်းများ (Things) ကို ရှာဖွေမှတ်သားတာမျိုး၊ စာသားထဲမှာ နာမည်တူပြီး အဓိပ္ပာယ် ကွဲပြားနိုင်တဲ့ စကားလုံးတွေ ပါလာတဲ့အခါ၊ ကွန်ပျူတာက ရှေ့နောက်စာသား (Context) ကို ကြည့်ပြီး "တကယ့် အစစ်အမှန် မည်သည့်အရာကို ဆိုလိုခြင်းဖြစ်သည်" ဆိုတာကို အတိအကျ ဝေခွဲဆုံးဖြတ်တာမျိုး။

**Example:** If a text mentions "Michael Jordan," the system must decide if it refers to the famous basketball player or the famous Machine Learning expert based on the context.

---

## 5. Knowledge Graphs:

**What it is:** Creating a web of connected information by extracting data from many different sources, such as Wikipedia.

    မတူညီတဲ့ ဒေတာရင်းမြစ် ပေါင်းစပ်မြောက်မြားစွာ (ဥပမာ - Wikipedia, news website၊ databaseများ) ဆီကနေ အချက်အလက်တွေကို နည်းပညာအရ ဆွဲထုတ်ပြီး၊ ၎င်းတို့အချင်းချင်း ချိတ်ဆက်နေတဲ့ "သတင်းအချက်အလက် ကွန်ရက်ကြီး (A web of connected information)" တစ်ခုအဖြစ် ဖန်တီးတည်ဆောက်တာ ဖြစ်တယ်။ 

    ဉပမာ - Google က သုံးနေတဲ့ Knowledge Graph ကြောင့် ငါတို့က "Aries" လို့ ရှာလိုက်ရင် ၎င်းနဲ့ သက်ဆိုင်တဲ့ နေ့စွဲတွေ၊ ဗေဒင်အယူအဆတွေ၊ Element (Fire) စတာတွေကို ကွန်ရက်ချိတ်ဆက်ပြီး တစ်ပြိုင်နက်တည်း ပြသပေးနိုင်တာ ဖြစ်တယ်။

---

## Summary

Data mining isn't just about spreadsheets; it’s about teaching computers to read and understand text (Text Mining), detect emotions (Sentiment Analysis), and even tell the difference between two people with the same name. It also uses high-tech tools like Deep Learning and Cluster Analysis to handle almost any kind of complex information.
