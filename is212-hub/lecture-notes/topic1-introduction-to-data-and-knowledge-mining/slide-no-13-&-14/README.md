# Class/Concept Description (slide13 and 14 - 8.6.2026)

In data mining, we often want to describe specific groups of information in a way that is easy to understand. We refer to these groups as classes or concepts. For example, a class could be "High-Income Earners" or "Frequent Shoppers."

    Data miningလုပ်တဲ့အခါ ဒေတာအုပ်စုအစိမ်းလိုက်ကြီးကို ကြည့်မယ့်အစား "High-Income Earners (လစာမြင့်မားသူများ)" သို့မဟုတ် "Frequent Shoppers (မကြာခဏ ဝယ်ယူတတ်သူများ)" စသဖြင့် လူတွေ နားလည်ရလွယ်ကူတဲ့ Classes / Concepts (အုပ်စုများ/အယူအဆများ) အဖြစ် ပြောင်းလဲပြီး နည်းလမ်း ၂ မျိုးနဲ့ ပုံဖော်လေ့ရှိတယ်။

There are two main ways to describe these groups:

---

## 1. Data Characterization (Summarising the Group)

**What it is:** This is a way of summarising the general features or characteristics of the specific group you are interested in, which is called the target class.

    မိမိလေ့လာချင်တဲ့ ပစ်မှတ်ဒေတာအုပ်စု (Target class) တစ်ခုတည်း ရဲ့ ယေဘုယျ ထူးခြားချက်တွေ၊ ဂုဏ်သတ္တိတွေကို စုစည်းပြီး အနှစ်ချုပ် တင်ပြတာ ဖြစ်တယ်။

**The Goal:** To give a high-level overview of what that group looks like

---

## 2. Data Discrimination (Comparing Groups)

**What it is:** This is a comparison of your target class against one or more "contrasting classes".

    မိမိရဲ့ ပစ်မှတ်ဒေတာအုပ်စု (Target class) ကို အခြား ဆန့်ကျင်ဘက် အုပ်စုတစ်ခု သို့မဟုတ် တစ်ခုထက်ပိုသော အုပ်စုများ (Contrasting classes) နှင့် ယှဉ်တွဲပြီး ကွဲပြားခြားနားချက်ကို ရှာဖွေတာ ဖြစ်တယ်။

**The Goal:** To find the features that make your target group different from others

---

## 3. How They are Used

A data mining system can provide descriptions using:

**Characterization only:** Just describing the target group.

    ဥပမာ - "ကျွန်မတို့ ဘဏ်ရဲ့ Credit Card အလွဲသုံးစားလုပ်ခံရတဲ့ (Fraud) ဖြစ်စဉ်တွေမှာ ဘယ်လို လက္ခဏာတွေ ရှိလဲ" ဆိုတာမျိုး

**Discrimination only:** Just showing the differences between groups.

    ဥပမာ - "ပုံမှန်ငွေလွှဲမှု (Normal Transactions) နဲ့ လိမ်လည်ငွေလွှဲမှု (Fraud Transactions) ကြားက ကွဲပြားချက်တွေက ဘာလဲ" ဆိုတာမျိုး

**Both:** Providing a complete picture by both summarising the group and showing how it differs from others

    ပစ်မှတ်အုပ်စုရဲ့ ထူးခြားချက်တွေကိုလည်း အနှစ်ချုပ်ပြသလို၊ ၎င်းအုပ်စုက အခြားအုပ်စုတွေနဲ့ ဘယ်လို ကွဲပြားနေလဲဆိုတာကိုပါ တစ်ပြိုင်နက်တည်း ယှဉ်ပြတာမျိုး

---

## Summary

Think of Characterization like writing a biography of one person (their hobbies, height, and job). Think of Discrimination like comparing two different people to see who is taller or who has a different job. To really understand a "class" of data, it helps to do both.
